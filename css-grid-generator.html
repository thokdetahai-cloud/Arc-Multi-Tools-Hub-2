About CSS Grid Generator

The CSS Grid Generator helps you build 2D CSS Grid layouts visually. Set the number of columns and rows, choose column templates (equal fractions, auto, fixed px, or responsive minmax), and adjust gap and alignment. The live preview shows the grid in action. Use the Flexbox Generator for 1D layouts.

Frequently Asked Questions

1fr means one fractional unit of available space — all columns with equal fr take equal shares. px columns have a fixed width. Mixing them (e.g., 200px 1fr 1fr) creates fixed + flexible layouts.

minmax(100px, 1fr) sets a minimum of 100px and maximum of 1fr (takes remaining space). This makes columns responsive — they never shrink below 100px but grow to fill space.
