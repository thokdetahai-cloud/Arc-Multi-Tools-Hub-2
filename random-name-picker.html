About Random Name Picker

The Random Name Picker selects one or more random names or items from a list you provide. Perfect for giveaways, raffle draws, team picking, assigning tasks, classroom exercises, and any decision that benefits from randomness. Enable 'No repeats' to ensure each name can only be picked once.
