About Border Radius Generator

The Border Radius Generator lets you control each corner of a CSS element independently or lock them all together. Use presets for common shapes like Square, Rounded, Pill, and Leaf. The live preview shows exactly how the element will look. CSS order is: top-left, top-right, bottom-right, bottom-left.

Frequently Asked Questions

CSS border-radius follows: top-left, top-right, bottom-right, bottom-left (clockwise from top-left). Shorthand with 2 values: top-left/bottom-right, top-right/bottom-left.

Using 50% creates a circle (for square elements) or ellipse. Values are relative to the element's dimensions, making percentage radius responsive.
