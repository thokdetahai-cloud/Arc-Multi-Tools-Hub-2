About PDF Metadata Viewer

The PDF Metadata Viewer extracts embedded metadata from PDF files — title, author, subject, creator, producer, creation date, page count, and encryption status. All processing happens locally in your browser using the FileReader API. Your PDF is never uploaded to any server.

Frequently Asked Questions

PDF metadata is embedded information about the document — its title, author, creation software, dates, and more. It's stored in the PDF file itself, separate from the visible content.

Not all PDFs include complete metadata. Many PDFs exported from simple tools only include basic fields. Scanned PDFs or older files often have minimal or no metadata.
